# daily-tasks
Коллекция задач по Java Core для отработки навыков.

## Темы
- **Коллекции:** List, Set, Map, Queue, итераторы
- **Функциональный стиль:** Stream API, лямбды, Optional, метод-ссылки
- **Многопоточность:** Thread, Runnable, synchronized, wait/notify
- **Исключения:** Иерархия, кастомные исключения, `try-with-resources`
- **Даты и время:** `java.time` API (LocalDate, LocalDateTime, ZonedDateTime, Instant), форматирование, вычисления (Duration/Period)
- **Файлы и I/O:** `java.nio.file` (Path, Files, StandardOpenOption), чтение/запись текстовых и бинарных данных, обход директорий (`Files.walk`/`Files.list`), работа с кодировками и `try-with-resources`
